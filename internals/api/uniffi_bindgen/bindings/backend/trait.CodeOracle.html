An object to look up a foreign language code specific renderer for a given type used. Every Type referred to in the ComponentInterface should map to a corresponding CodeType.

The mapping may be opaque, but the oracle always knows the answer.

In adddition, the oracle knows how to render identifiers (function names, class names, variable names etc).
