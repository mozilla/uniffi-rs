Expand description

Represents an attribute parsed from UDL, like [ByRef] or [Throws].

This is a convenience enum for parsing UDL attributes and erroring out if we encounter any unsupported ones. These don’t convert directly into parts of a ComponentInterface, but may influence the properties of things like functions and arguments.
